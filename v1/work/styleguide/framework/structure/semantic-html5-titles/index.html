Semantic HTML5 Titles

Description

The HTML5 structure should outline perfectly in the HTML5 Outliner. For that all content is wrapped into semnatic HTML5 elements like <section>, <article>, <nav> etc. The title of these elements are necessary to display well in the Outliner but are hidden on the UI.
