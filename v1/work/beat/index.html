Description

Beat is a blog about design and code. It's lazy yet meta. With focus on that few stories which are shaping the front-end a year.

The design is clean. Typography and form follows the modern style while decorations are postmodern.

Beat is a static site created with JSON, Gulp, Webpack. It's an experiment in the Headless CMS scene.
