The idea is to create simple templates which display only the post / page content which can be full with custom Gutenberg blocks.

For example, the `Home` page could contain the following blocks:

  • Header
  • Hero
  • 3 Features
  • News
  • Footer

Or the `Archives` page could contain these blocks:

  • Header
  • List of posts
  • Footer
