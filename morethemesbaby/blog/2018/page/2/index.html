More Themes Baby is in the loop

Post featured image

Post content for Standard Post format

The Front-End Tooling Survey 2018 Results is out and we are glad to see all technologies we use to create themes are in line with the majority.

  • SASS / SCSS — 65.33%
  • No CSS framework (33.16%) and Custom CSS framework (15.17%) vs. Bootstrap (34.69%)
  • BEM as CSS methodology and naming scheme — 38.85%
  • Autoprefixer  for cross browser compatibility — 57.45%
  • Flexbox and Grid as emergig CSS features — 68.04% and 17.95%
  • Gulp as task runner — 29.70%, second place after NPM

On Log Lolla Theme

Post content for Standard Post format

As I’ve written earlier I’ve lost contact with WordPress around ten years ago. This means I’ve never used it for my own content, instead I’ve used Tumblr, Twitter, etc and my self-made blogs to do the job.

Now I find joy in posting everything to this blog running the Log Lolla theme. I’ve always pursued to create knowledge from my everyday learning and reading. Up until now this knowledge was scattered on many external services … now I’ve got the heureka and the mission: create a theme which is the centrum of all this activity.

After maturing Log Lolla will be the single destination to collect my thoughts, inspiration, links to learn, development timeline, and all else going around my mind.

One day, I hope, it will be also yours 🙂
