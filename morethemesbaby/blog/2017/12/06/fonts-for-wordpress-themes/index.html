Fonts For WordPress Themes

Post featured image

Post content

Everything goes into a WordPress.org theme — fonts, images, icons, etc — should have a GPL compatible license.

Like Google Fonts with their 800+ specimens freely available. No wonder majority of themes are built on these fonts.

Here at Mo’ Themes Baby we would like to check if there are any other sources of free fonts available and use them — only if they match the Google Fonts quality.

We don’t need tons of free fonts. We need a sans-serif, a serif, a display, and a monospace font four our next themes. Each of them with basic styles and weights.

Open Foundry

  • Description: 30 carefully selected fonts, mostly for display
  • Link: http://open-foundry.com/hot30

The Northern Block

  • Description: A good few rare fonts, mostly for display
  • Link: www.thenorthernblock.co.uk

Font Squirrel

  • Description: Probably the best list of free fonts for commercial use. It overlaps heavily with Google Fonts but there are plenty hidden treasures.
  • Link: https://www.fontsquirrel.com/

FontStruct Gallery

  • Description: Make your own font, or browse fonts created by the community
  • Link: https://fontstruct.com/gallery

Inter UI

  • Description: A full font family with a very standard look
  • Link: https://rsms.me/inter/
  • License: SIL Open Font

Swiss Typefaces

  • Description: Very affordable and uttermost modern font families
  • Link: https://www.swisstypefaces.com/

AtF Spark

  • Link: https://aftertheflood.co/projects/atf-spark/
  • License: SIL Open Font

Other interesting stuff

  • Klim Type Foundry, Tight Type: very modern typefaces, with a web license based on monthly pageviews
  • Lost Type, free fonts for personal use
  • On snot and fonts, the most comprehensive catalog about typography
  • Typography Supply, a list of tools for web typography
  • Color Fonts, the next big thing
  • Variable fonts, A List Apart article
  • Anymography, animated typography
