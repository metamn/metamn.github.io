TTF

Open Society Institution's Think Thank Fund was looking for a new agency for their future infographics, print and web projects.

we've put together an expert team — Isabel Meirelles on infographics, Botond Reszegh on print — and come up with an unique, all in one solution: websites which print.

responsiveness

rich interactions

rich interactions

by nature TTF projects are built on rich interactive components. we've did an extensive research, collected and tested all state-of-art open source libraries which are responsive, and use some of then when it was necessary.

focus on print

focus on print

the main idea was to create the website in such way it can be printed anytime. we've carefully choose only those design elements which look the same on screen and paper.

a website which prints

a website which prints

it turned out making a website to print is ... nothing more than designing it for a new device, which is the paper in this case.

using responsive design techniques and choosing the right layout, colors, typographic and design elements any website can be made to be printed.
